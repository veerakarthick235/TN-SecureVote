# TN SecureVote

Secure Online Voting SaaS Platform for Tamil Nadu

## 🚀 Overview

TN SecureVote is a production-grade, end-to-end verifiable online voting
system designed to eliminate fraud, ensure anonymity, and provide
transparent auditing.

## 🔐 Features

-   End-to-end verifiable voting
-   Blockchain-based tamper-proof storage
-   Blind signature anonymity
-   Zero-Knowledge Proof verification
-   Demo mode included
-   Tamil + English bilingual UI

## 🧩 Tech Stack

-   Frontend: React + Vite + TypeScript
-   Backend: FastAPI (Python)
-   Blockchain: Hyperledger Fabric (simulated)
-   Database: PostgreSQL / SQLite
-   AI: Fraud detection (Isolation Forest, LSTM)

## ⚙️ Installation

### Backend

``` bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

``` bash
cd frontend
npm install
npm run dev
```

## 🌐 Access

-   Frontend: http://localhost:5173
-   Backend: http://127.0.0.1:8000/docs
